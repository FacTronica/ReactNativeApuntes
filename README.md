# React Native con Expo

### Crear APP Vacia
npx create-expo-app@latest webview --template blank

### Instalar Web
npx expo install react-native-web react-dom @expo/metro-runtime

### Ejecutar una APP Web
npm run web

### Instalar Cliente Expo
npm install -g eas-cli
